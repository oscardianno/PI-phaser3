Esta es la estructura del proyecto:  
- 1 carpeta que se llame "js"  
- 1 carpeta que se llame "css"  
- 1 carpeta que se llame "assets"  
- 1 archivo index.html  
- 1 archivo index.php (este llama a index.html)
